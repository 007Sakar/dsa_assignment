# Reverse Traversal of Singly Linked List

## Objective
To traverse and display a singly linked list in reverse order.

## Data Structure Used
A singly linked list implemented using a structure with:
- Data field
- Pointer to next node

## Functions Implemented
- `printReverse()`  
  Uses recursion to print the linked list in reverse order.

## main() Overview
- Creates a linked list dynamically.
- Inserts data into the list.
- Calls the reverse traversal function.
