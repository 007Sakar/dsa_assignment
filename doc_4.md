# Doubly Linked List Implementation

## Objective
To implement a doubly linked list and perform insertion and deletion operations.

## Data Structure Used
A doubly linked list where each node contains:
- Data
- Pointer to previous node
- Pointer to next node

## Functions Implemented
- `insertAfter()`  
  Inserts a new node after a given node.

- `deleteNode()`  
  Deletes a specified node from the list.

## main() Overview
- Creates a doubly linked list.
- Inserts nodes after a given node.
- Deletes a node.
- Displays the final list.
