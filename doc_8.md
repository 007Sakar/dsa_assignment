# Sorting Random Numbers with Performance Count

## Objective
To generate random numbers, sort them, and count comparisons and swaps.

## Data Structure Used
An array to store integer values.

## Sorting Algorithm Used
Bubble Sort.

## Variables Used
- `comp` to count comparisons
- `swap` to count swaps

## main() Overview
- Takes number of elements from user.
- Generates random numbers.
- Displays numbers before sorting.
- Sorts using bubble sort.
- Displays sorted numbers and operation counts.
