# Min Heap and Max Heap Construction

## Objective
To build both a min heap and a max heap from an unsorted array.

## Data Structure Used
An array is used to represent the heap.

## Functions Implemented
- `heapifyMin()`  
  Maintains the min heap property.

- `heapifyMax()`  
  Maintains the max heap property.

## main() Overview
- Initializes an unsorted array.
- Builds a min heap from the array.
- Builds a max heap from the array.
- Displays both heaps.
